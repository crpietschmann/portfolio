iPhone style flicking support using JavaScript

This page uses JavaScript to implement iPhone style flicking to scroll up/down the page.
This code does not work properly in Firefox. I didn't bother getting it to work in Firefox, since this is a UI feature that is really only meant to be used with Mobile devices (such as the Apple iPhone).

How to use:
  • On a PC: Click and drag your mouse real quick and let go of the mouse button shortly after dragging.
  • On the iPhone: Just flick the page with your finger.
If you flick real quick it'll keep scrolling all the way to the top or bottom of the page.
To stop it from auto-scrolling, just click or tap the page.

Browsers Supported:
  • Apple Safari 3.0 (I only tested the Windows version since I don't have a Mac.)
  • Applie iPhone (I haven't tested this, could someone with an iPhone please let me know if it has any problems?)
  • Microsoft Internet Explorer 7
