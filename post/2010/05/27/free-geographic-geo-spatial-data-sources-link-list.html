Free Geographic / Geo-Spatial Data Sources Link List

May 27, 2010  • GIS  • Data

This page lists a bunch of different geographic / geo-spatial data sources that provide FREE data to be consumed. I haven’t really found any resource link list like this, so I thought I’d start one.

Maps / Charts

http://www.nauticalcharts.noaa.gov/ - NOAA’s Office of Coast Survey — USA http://www.nauticalcharts.noaa.gov/mcd/Raster/download_agreement.htm

http://www.nauticalcharts.noaa.gov/mcd/Raster/index.htm - Raster Navigational Charts: NOAA RNCs

Cartographic Boundary Data

http://www.census.gov/geo/www/cob/bdy_files.html - State, County, ZipCode, School District, etc. in USA

Points of Interest

http://geonames.usgs.gov/domestic/download_data.htm - Domestic Name - State and Topical Gazetteer Download Files <– Points of Interest all over USA

http://www2.ed.gov/parents/schools/find/edpicks.jhtml - ED.GOV - Find Schools

http://nces.ed.gov/globallocator/ - Search for Schools, Colleges and Libraries – USA

http://dpi.state.wi.us/districts.aspx - List of Wisconsin School Districts

http://wise.dpi.wi.gov/wisedash_downloadfiles - Wisconsin WISEdash Data File by Year

http://wise.dpi.wi.gov/about-data

Other Data

http://earthquake.usgs.gov/earthquakes/feed/v1.0/ - USGS Real-time Feeds & Notifications

http://www.census.gov/geo/maps-data/data/partnership.html - US Census Partnership Shapefiles

http://www.census.gov/geo/maps-data/index.html - US Census Geography Maps & Data

http://www.census.gov/geo/maps-data/data/gazetteer2010.html - 2010 Census Gazeteer Files

ftp://ftp2.census.gov/geo/tiger/TIGER2012/ - US Census 2012 Tiger Data

http://databases.about.com/od/access/a/zipcodedatabase.htm - Free USA Zip Code Data

ftp://ftp2.census.gov/geo/tiger/TIGER2009/ - US Census 2009 Tiger Data

ftp://ftp2.census.gov/geo/tiger/TIGER2008/ - US Census 2008 Tiger Data

http://www.census.gov/geo/www/gazetteer/gazette.html

http://www2.census.gov/census_2000/datasets/

Free ShapeFile Downloads

http://openstreetmapdata.com/data/land-polygons

http://thematicmapping.org/downloads/world_borders.php

http://www.naturalearthdata.com/downloads/10m-cultural-vectors/

http://download.geofabrik.de/
